###### opGrandes

#### description

este programa em C realiza operações manipulando números que excedem os limites padrões, como `int` ou `long long`, com base em strings.

#### features

  - soma, subtração e multiplicação de strings;

#### how to run

    1. gcc -o opGrandes opGrandes.c
    2. ./opGrandes
    3. <número de casos> <número1> <número 2> <operação (1, 2 ou 3)>

            e.g.:
    2. ./opGrandes
    3. 1 121521913456789 33324211987654321 2
